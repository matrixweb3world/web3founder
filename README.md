# Web3-Founder

Web3 去中心化应用程序开发项目

## 功能模块

- **Discover**: Twitter KOL 监控与 AI 分析系统
  - KOL 监控仪表盘
  - 情绪分析图表
  - 最新推文列表
  - AI 解读分析

## 开发指南

详细的开发指南，包括热更新和模拟器使用说明，请参阅：
[开发指南](./README_DEV.md)

## 快速启动

```bash
# 启动模拟器并运行应用
./web3-founder/run_simulator.sh

# 启用热更新（监控文件变化并自动重构）
./web3-founder/hot_reload.sh

# 手动构建和安装应用（当热更新不工作时）
./web3-founder/manual_build.sh
```

## 环境要求

- iOS 14.0+
- Xcode 15.0+
- Swift 5.9+
- macOS Sonoma 14.0+
